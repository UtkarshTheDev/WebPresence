Welcome to Web Presence

Share your browsing activity on Discord with rich presence

About Web Presence

Web Presence shows your current website in your Discord profile. When you browse websites like YouTube, GitHub, or Twitter, your Discord profile will display what you're doing with custom icons for each site.

This extension works together with a local server that connects to Discord. Let's make sure everything is set up correctly!

Setup Guide

1 Make Sure Discord is Running

Web Presence connects to the Discord desktop app (not the web or mobile version).

  • Make sure Discord desktop app is open
  • Check Discord Settings > Activity Status > Make sure "Display current activity as a status message" is ON
2 Install the Server

Next, you need to install the Web Presence server. Open your terminal or command prompt and run:

npm install -g webpresence
Make sure you have Node.js installed (version 16 or higher).
3 Start the Server

Start the Web Presence server in daemon mode so it runs in the background:

webpresence start -d webpresence status
Using daemon mode (-d) lets the server run in the background so you can close your terminal window!

Optional: Configure Web Presence to start automatically when your computer boots:

webpresence autostart --enable
This ensures Discord presence is always available without manual intervention!
4 Enable Web Presence

Now that everything is set up, you can enable Web Presence:

  • Click the Web Presence icon in your browser toolbar
  • Toggle the switch to enable Discord presence
  • Start browsing - your activity will show in Discord!

Troubleshooting

If you're having issues, try these common solutions:

Discord Not Connecting
  • Restart Discord completely (close from system tray)
  • Restart the Web Presence server:
    webpresence stop webpresence start -d
Server Not Running
  • Check server status:
    webpresence status
  • If not running, start it again:
    webpresence start -d
